#What is JSX?
#####Answer
JSX is a HTML or XML like syntax. It stands for Javascript XML. It is syntax extension for Javascript.
It is not HTML inside Javascript. It is not a part of React. Without of JSX also we can write our react code but it 
will be very hard to read.
Javascript engine does not understand JSX.
